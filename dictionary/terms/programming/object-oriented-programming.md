# Object-Oriented Programming (OOP)

## Description

نموذج برمجي يعتمد على مفهوم الكائنات التي تحتوي على بيانات ووظائف. يركز على التجريد والتغليف والميراث والتعددية لتنظيم الكود بطريقة قابلة لإعادة الاستخدام.
