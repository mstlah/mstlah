# Compiler

## Description

برنامج يترجم الكود المصدري المكتوب بلغة برمجة عالية المستوى إلى لغة منخفضة المستوى يمكن للحاسوب تنفيذها مباشرةً، مثل لغة التجميع أو الكود الثنائي.
